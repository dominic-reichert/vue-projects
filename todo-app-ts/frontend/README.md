# TodoApp with TypeScript and Vue's Composition Api

<p>Small Todo-App project to play around with TypeScript and to get to know Vue's composition api.</p>

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files
